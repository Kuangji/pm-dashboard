## 贡献指南

1. 在 `drafts/publish/documents/` 目录下添加 Markdown 文件
2. 更新 `drafts/publish/manifest.json` 配置导航
3. 运行 `npm run sync` 同步内容
4. 访问 `http://localhost:3000/docs/` 查看效果
