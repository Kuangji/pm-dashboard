# 标准搜索自然语言输入 · Input v1

> 归属版本：`v6.3.9`

## 背景

现有标准搜索已经有稳定的 schema 参数与结果页，但用户仍需自己理解关键词和筛选器如何组合，导致自然语言需求无法自然进入现有搜索体系。

## 目标

让用户在标准搜索中显式切换到自然语言输入形态，用一句话构建或调整当前 schema 参数，并继续使用现有标准搜索结果页。

## 本期包含

- `关键词 / 自然语言` 输入形态切换
- 自然语言轻解析
- 回填现有关键词 / 筛选参数体系
- 结果页顶部 `输入框 -> 当前生效条件 -> 折叠历史`
- 解析失败内联提示

## 本期不包含

- 智能精选
- 标准搜索到智能精选的桥接
- 解释层、diff 层、聊天流

## 成功标准

- 用户能在标准搜索里切到自然语言输入
- 自然语言能回落到现有 schema 参数
- 成功后仍留在标准搜索结果页
- 失败时旧结果继续保留
