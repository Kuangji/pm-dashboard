# 标准搜索自然语言输入 · Design v1

> 归属版本：`v6.3.9`

## 一、设计原则

- 让用户感觉是“标准搜索增强”，而不是进入另一个产品
- 尽量保持现有搜索页稳定
- 明确区分 `关键词模式` 与 `自然语言模式` 的展示差异

## 二、输入区

### 2.1 关键词模式

- 保持现有搜索框和筛选器
- 关键词胶囊继续留在框内

### 2.2 自然语言模式

- 输入框更长，承载一句完整需求
- 当前生效条件固定展示在输入区下方
- 不再把关键词留在输入框里

## 三、成功后结果页

- 顶部变成紧凑工作台
- 结果区仍然是标准搜索列表
- 历史是折叠时间线，不是聊天记录
- 用户继续输入下一句时，仍在标准搜索心智内

## 四、失败态

- 提示贴在输入框下
- 不切独立错误页
- 旧结果继续保留

## 五、版本边界

- `v6.3.9` 不在标准搜索产品 UI 中展示智能精选入口
- 所有“去智能精选”的桥接讨论移到 `v6.3.10`
