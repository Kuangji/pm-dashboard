# 单频道相似网红搜索 · Spec v1

> 归属版本：`v6.3.9`

## 一、入口结构

`相似网红` 保持独立入口，不并入：

- 标准搜索
- 智能精选

## 二、新默认流程

### 2.1 首屏输入

输入来源：

- `search_select`
- `url`
- `handle`

规则：

- 默认是一个智能混合框
- 搜索关键词必须先选中一个明确频道
- URL / `@handle` 在显式提交时解析
- 主按钮只有在形成有效种子后才可用

### 2.2 有效种子态

- 顶部输入壳层切为频道卡展示态
- 展示头像、频道名、handle、平台
- 提供 `更换频道`

### 2.3 执行中

- 必须有独立执行中态
- 清楚显示当前种子频道
- 输入区暂时锁定

### 2.4 结果态

- 结果保持普通相似结果列表
- 顶部继续保留当前种子壳层
- 更换频道时，旧结果在新提交前继续保留

## 三、旧流程保留

- 页内保留次级 Tab：`收藏夹相似搜索`
- 但不抢默认入口主视觉

## 四、验收标准

- 默认首屏是单频道输入，不是收藏夹
- 新结果形态仍是普通列表，不误入智能精选批次流
- 旧流程仍可访问
