# 单频道相似网红搜索 · Input v1

> 归属版本：`v6.3.9`

## 背景

现有相似网红默认依赖收藏夹输入，召回逻辑基于标签聚合与重合度计算，起手重、心智不直观。

## 目标

把默认入口改成更轻的 `单频道输入 -> 多模态向量召回`，同时保留旧收藏夹方案作为次级入口。

## 本期包含

- 智能混合框输入
- 单频道种子识别
- 执行中态
- 多模态相似结果列表
- 旧收藏夹次级 Tab

## 本期不包含

- 智能精选
- 批次消费
- AI 会话流

## 成功标准

- 不依赖收藏夹也能直接开始相似搜索
- 用户能理解“输入频道 -> 找相似频道”的链路
- 老流程仍可达但不再是默认入口
