# 单频道相似网红搜索 · Design v1

> 归属版本：`v6.3.9`

## 一、设计目标

- 默认入口更轻
- 起手更明确
- 结果仍然保持专业搜索工具感

## 二、首屏

- 一个智能混合框
- 明确的主按钮：`开始找相似网红`
- 清楚提示支持搜索、URL、`@handle`
- 次级 Tab 展示旧收藏夹流程

## 三、种子识别后

- 输入壳层从编辑态切到频道卡展示态
- 明确这是“当前频道种子”
- 提供 `更换频道`

## 四、执行中与结果

- 执行中显示当前种子并锁定输入区
- 结果页仍然是普通相似结果列表
- 顶部允许更换频道，但不引入批次消费或 AI 工作台感

## 五、版本边界

- 本版本只处理相似网红默认流程替换
- 不与智能精选做桥接或合流
