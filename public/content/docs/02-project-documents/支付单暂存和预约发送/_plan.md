#### 03-mindmap-part1.png · 支付流程脑图

模块标题：**聚星营销：推广费用模块（也就是支付）**，主干节点为「流程概览」。

**背景约束（顶部三条）**：
- 只支持向**境外银行账号汇款**的方式
- 不支持向国内银行账号付款
- 不支持包括 PayPal 在内的**第三方支付**方式

**完整流程节点**：

| 步骤 | 名称 | 关键细节 |
|------|------|---------|
| 0 | 背景 | 聚星营销推广任务流程中，对网红支付推广佣金的模块 |
| 1 | 开户 | 广告主与聚星平台签订营销服务框架合同，并同意用户协议 |
| 2 | 充值 | 广告主向平台账号充值**美金** |
| 3 | 开单 | 广告主建立营销支出单发起支付（详见下方字段拆解） |
| 4 | 告知 | 合作网红拿到线上推广协议并签署 |
| 5 | 审核 | 聚星平台审核支出单（初期人工，后期自动） |
| 6 | 转账 | 聚星平台向收款方发起转账 |
| 7 | 到账 | 转账到达目的地银行 |
| 8 | 管理 | 查询账目等 |

**步骤 3「开单」字段拆解**：

- **网红频道**：输入聚星详情页或对应平台频道页即可查询选择；**仅支持平台已收录频道**
- **任务信息**：
  - 必须绑定一个平台的线上任务（Campaign），没有的可以当场新建，但必须准确填写营销信息
  - 选择后自动读取任务信息到支出单，并可修改
  - 一旦任务下绑定了支付单，则上述任务信息将**不可被修改**
- **收款人信息**：国家、币种、银行名称、账号、账户名、地址等
  - 随国家不同要求不同，以线上为准，有预览
  - 由合作渠道（空中云汇）规定，不能调整
- **金额计算**：
  - 填入收款金额（目标货币），自动计算对应支出美金金额
  - 自动计算聚星**服务费**：标准 5%，早期临时方案为 0 服务费；量大可谈阶梯计费
  - 自动计算**转账手续费**：以收款国家、币种及转账方式决定；建议选本地转账（Local），手续费低、时效好（当天或 T+1）
  - **总扣除余额 = 支出美金 + 服务费 + 转账手续费**
  - 正式转账前按预估数字**冻结余额**
  - 开始转账时以银行行汇率计算为准，**多退少补**
  - 最低起付金额 **$50**；账户最少余额 **$50**（防止击穿）
- **生成支出单**：正式转账前仍需两步（合同签署 + 审核）

**步骤 5「审核」细节**：
- 检查点：余额足够支付 + 合作内容无不当信息
- **非**检查点：网红收款信息准确性

```
初始
 └─→ A（暂存支付单，等待合同签字完毕）

      └─→ A1（合同签字完成，支付审核中）
           ├─→ A3（广告主在审核完成前取消支付）【终止】
           ├─→ C（审核未通过）【终止】
           └─→ B（审核通过，等待支付通道处理）
                └─→ B1（支付通道处理中）
                     ├─→ B2（支付通道处理失败）【终止】
                     └─→ B4（已转入收款行）
```
