# 支付单暂存和预约发送

> 功能模块入口文档 · VDR 工作流

## 功能概述

在现有支付管理模块基础上，新增**支出单暂存**和**预约发送**能力，解决品牌方在创建支出单时无法保存草稿、无法指定未来时间发送的痛点。

## 所属模块

- 一级导航：合作管理
- 二级导航：支付管理 → 收支明细
