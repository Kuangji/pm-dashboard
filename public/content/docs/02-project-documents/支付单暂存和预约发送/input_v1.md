## 一、背景与动机

### 支付链路概述

NoxInfluencer 平台的境外网红支付链路如下：

**开户 → 充值美金 → 开单 → 网红签推广协议 → 平台审核 → 转账 → 到账**

关键约束：
- 仅支持境外银行账号汇款，**不支持**国内银行账号和 PayPal
- 总扣除余额 = 支出美金 + 服务费（5%）+ 转账手续费
- 转账前按预估金额**冻结余额**，实际转账后多退少补
- 最低起付金额 **$50**，账户最少余额 **$50**

### 支付单状态机

完整状态共 **10 个**，分四个阶段：

| 状态码 | 含义 | 阶段 |
|--------|------|------|
| A | 暂存支付单，等待合同签字完毕 | 创建阶段 |
| A1 | 合同签字完成，支付审核中 | 审核阶段 |
| A2 | 广告主在合同签字前取消【终止】 | 创建阶段 |
| A3 | 广告主在审核完成前取消【终止】 | 审核阶段 |
| B | 审核通过，等待支付通道处理 | 转账阶段 |
| B1 | 支付通道处理中 | 转账阶段 |
| B2 | 支付通道处理失败【终止】 | 转账阶段 |
| B4 | 已转入收款行 | 到账阶段 |
| C | 审核未通过【终止】 | 审核阶段 |
| D | 收款行业务异常，退款中 | 到账阶段 |
| D1 | 退款已到账【终止】 | 到账阶段 |

> 新功能作用位置：**功能一（暂存）** 作用在状态机之外（表单层）；**功能二（预约发送）** 插入在 **B → B1** 之间（Spec 中细化为 B-a → **B-b（预约等待中）** → B-c → B1，详见 spec_v1.md §四）。

### 现存问题

> 来源：真实用户反馈/投诉

- **无草稿机制**：创建支出单流程步骤较多，用户填写过程中若中断（如切换页面、意外关闭），已填内容全部丢失，需从头重填
- **无法延迟打款**：当前流程中，合同签字完成后系统即进入审核并自动触发转账，品牌方无法约定特定打款日期（如月结日、财务结款日），只能即时提交，控制权不在用户手中

## 三、核心功能方向

### 功能 1：支出单暂存（草稿）

**解决问题**：填单过程中断后内容丢失

**作用层**：表单层，独立于支付单状态机之外（不引入任何新的支付单状态）

**触发方式**（两种）：
1. **主动保存**：用户点击表单中的「保存草稿」按钮
2. **关闭提示**：用户关闭创建弹窗时，系统弹出提示「是否保存草稿？」，用户确认后保存

**草稿行为**：
- 草稿支出单在列表中可见，状态为「草稿」
- 草稿支出单支持继续编辑和最终提交

### 功能 2：预约发送

**解决问题**：合同签字锁定后，品牌方希望控制打款日期，不希望立即触发转账

**作用点**：在状态 **B（审核通过，等待支付通道处理）→ B1（支付通道处理中）** 之间插入定时等待

**设定时机**：预约时间在**创建支出单时**一并设定

---

**自动触发条件（AND 关系，两者同时满足时系统自动触发 B1）：**

1. ✅ 前置流程完成：合同签字完成 + 平台审核通过（已进入状态 B）
2. ✅ 预约时间到达

---

**场景一（成功路径）：**

合同先签完 → 审核通过（进入状态 B）→ 预约时间到达 → 系统自动触发 B1（转账开始）

**场景二（失败路径）：**

预约时间到达时，合同未签或审核未通过（尚未到达状态 B）→ 标记预约失败 + 触发用户提醒

---

**规则约束：**

| # | 规则 | 说明 |
|---|------|------|
| 1 | 预约时间最短限制 | 预约时间距创建时至少 **3 天**后 |
| 2 | 24h 签约提示 | 创建预约时，明确提示用户「请在预约时间的 **24 小时前** 完成网红签合同，否则审核可能无法完成，导致预约失败」 |
| 3 | 48h 前提醒 | 预约时间 **48 小时前**向广告主发送提醒（提醒形式待定，站内信/邮件/短信待 Spec 阶段确认） |

## 四、待厘清的问题

> 状态说明：✅ 已确认 · ⬜ 待 Spec 阶段确认

### 已确认

| 问题 | 确认结论 |
|------|---------|
| ✅ 两个功能是否同期上线 | 是，同期上线 |
| ✅ 暂存触发方式 | 主动点击「保存草稿」+ 关闭弹窗时系统提示 |
| ✅ 预约发送的核心场景 | 合同签字后控制打款日期，不立即触发转账 |
| ✅ 预约触发点 | B → B1 之间，AND 条件：B 达到 + 时间到达 |
| ✅ 预约时间最短限制 | 创建时起至少 3 天后 |
| ✅ 失败路径处理方式 | 标记预约失败 + 触发用户提醒 |
| ✅ 48h 前提醒机制 | 预约时间 48h 前向广告主发提醒（形式待定） |
