# 发版公告｜v6.2.5 支付管理支持支出单暂存与预约发送

尊敬的用户：

在 **v6.2.5** 版本中，我们在「合作管理 > 支付管理」中上线了两项新能力：**支出单暂存（草稿）** 和 **预约发送**，帮助您更灵活地安排付款流程。

本次更新后，您在创建营销支出单时，如果临时需要中断填写，可以先保存为草稿，稍后继续编辑；如果希望在指定日期发起打款，也可以选择预约发送，更方便配合合同签署进度和财务安排。

## 本次更新内容

### 1. 支出单支持暂存为草稿

- 创建营销支出单时，可将当前内容保存为草稿，稍后继续填写
- 若您在填写过程中关闭弹窗，系统也会在合适场景下提醒您是否保存草稿
- 草稿会展示在支付管理列表顶部，方便快速找到并继续编辑
- 草稿提交成功后会自动消失，不需要手动删除

### 2. 支持预约发送

- 提交营销支出单时，您可以选择「立即发送」或「预约发送」
- 选择预约发送后，可指定具体的发送日期和小时
- 当支付单完成前置流程，且到达您设置的预约时间后，系统会自动发起后续转账流程

## 如何使用

### 保存草稿

1. 进入「合作管理 > 支付管理」
2. 创建营销支出单
3. 选择网红频道后，即可点击「保存草稿」
4. 后续可在列表中找到对应草稿，点击「继续编辑」恢复之前填写的内容

### 预约发送

1. 创建营销支出单并完成必要信息填写
2. 在提交时选择「预约发送」
3. 选择希望发送的日期和小时
4. 确认提交后，系统将在满足条件时按预约时间自动处理

## 温馨提示

- 草稿仅保存表单填写内容，**不会保存发送时间**
- 预约发送时间需至少晚于当前时间 **72 小时**
- 建议您尽量在预约时间前 **24 小时** 完成合同签署，以免影响后续审核和打款安排
- 若草稿进入继续编辑状态，已选择的网红频道不支持直接修改；如需更换，请新建支出单
- 若预约时间已到，但合同签署或审核尚未完成，系统会通过站内信提醒您，并引导您继续处理
