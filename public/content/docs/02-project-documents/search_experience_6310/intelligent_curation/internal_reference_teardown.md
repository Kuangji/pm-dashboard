# 智能精选 · 旧参考拆解

> 归属版本：`v6.3.10`

## 参考来源

1. 营销计划页旧 `智能精选`
2. `channel_search_agent_upgrade`

## 关键事实

- 营销计划页旧智能精选强绑定 `campaignId`
- 它不是可直接搬到频道搜索的全局能力
- 但可借鉴：
  - 独立模式感
  - 自然语言主输入
  - 任务执行中的过程感

## 本次设计影响

- 智能精选必须脱离营销计划上下文重定义
- 标准搜索桥接带入的是 `结果快照 + 来源条件`，不是若干 seed 字段
- 标准搜索与智能精选需要保持 session 独立，不自动双写
