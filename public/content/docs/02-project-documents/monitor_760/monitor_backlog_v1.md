# 内容监控项目：未来需求储备池 (Backlog)

## 1. 列表与管理增强

### 1.1 智能分组与聚合 (Smart Grouping)
- **来源**: 7.6.0 版本设计期间的灵感。
- **描述**: 当任务量大时，支持按“频道”或“标签/计划”进行聚合折叠。
- **价值**: 快速查看 KOL 总贡献或 Campaign 整体表现，减少列表冗余。
- **计划版本**: TBD

### 1.2 评论/反馈直接透出 (Comment Snippet)
- **描述**: 在列表行内直接透出最新热评或 AI 情感总结。
- **价值**: 无需进入详情页即可感知内容舆情。

### 1.3 自定义列表视图 (Custom Column Layout)
- **描述**: 支持用户保存自定义的表头配置（如 Twitter 视图、YouTube 视图）。

## 2. Twitter (X) 进阶监控

### 2.1 深度社交指标
- **引用推文 (Quotes)**: 独立展示二次创作数据（需评估数据接口可用性）。
- **互动构成比 (Engagement Mix)**: 分析“转推/点赞比”，区分内容的扩散性与认可度。

### 2.2 详情页高级可视化
- **Twitter 专属详情面板**: 包含阅读量爆发曲线、书签增长趋势等特化图表（待数据源稳定后实施）。
