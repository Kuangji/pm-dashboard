# 原始输入：Twitter (X) 监控支持 (Module 3)

> **公共上下文引用**: 参见 `monitor_common_context_v1.md`

## 1. 核心业务诉求
扩展内容监控对 Twitter (X) 平台的覆盖，实现基础数据的实时追踪与展示。

## 2. 核心功能点

### 2.1 基础监控指标 (X-Specific Metrics)
*   **核心指标 (仅在 Twitter 筛选下展示)**: 
    *   **阅读量 (Views)**: 代替传统的播放量。
    *   **书签 (Bookmarks)**: 反映高意向收藏数据。
    *   **转推 (Retweets)**: 合并显示直接转推。
*   **通用指标**: 点赞 (Likes)、回复 (Replies)。
*   **展示逻辑**: 
    *   当切换至“Twitter”平台时，列表表头动态更新指标项。
    *   在“全部”视图下，阅读量映射至“播放量”列。

### 2.2 内容解析与预览
*   **纯文字预览**: 若推文无图，缩略图显示“文字摘要卡片”（背景色 + 推文前 50 字）。
*   **多媒体支持**: 支持解析单图、多图预览及视频封面。
*   **结构化内容适配 (New)**:
    *   **推文串 (Threads)**: 列表标题旁增加“Threads”标识，数据展示该串首条推文的表现。
    *   **投票 (Polls)**: 解析出投票选项及其百分比，在详情页透出。
    *   **引用推文 (Quotes)**: 详情页需展示被引用推文的摘要（原博主、原正文）。

---
**执行备注**: 
- 自动追踪逻辑复用现有的 Facebook/TikTok 框架，无需额外设计。
- 引用推文及详情页深度图表由于数据限制，暂不在此版本实现。
