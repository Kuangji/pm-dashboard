## 核心事实

- Klear 自身定位是数据驱动的 influencer marketing platform。
- 官方搜索说明强调：
  - Klear 在搜索前先把 influencer 组织进 search index
  - 每个 influencer 会按 Topic 分类
  - Topic classification 由 AI 结合 hashtags、audience interests、network quality 等信号完成
- 官方营销指南明确写到 discovery 可设置的 filters 包括：
  - audience location
  - influencer / audience gender
  - audience age
  - influencer category
  - hashtags
- 官方还强调 `True Reach` 等 performance metric 用于后续 vetting

## 关键摘录

- Klear 的 AI 更像“搜索底层的智能分类与排序”，而不是显性的 chat assistant
- 它提供的是 `beautifully simple, remarkably smart` 的数据产品心智
- 搜索仍然明显依赖结构化筛选器
