# Modash · AI Search 原始信息

采集时间：2026-03-19

## 来源

- 官方产品页：<https://www.modash.io/features/influencer-discovery/ai-search>

## 核心事实

- Modash 将 `AI Search` 明确放在 `Discover` 能力之内，而不是独立外置工具。
- 官方主张是 “Describe it. Find it.”，直接强调“用自己的话找 creator”。
- AI Search 支持：
  - 自然语言搜索
  - 图片搜索
  - 在同一结果视图里看 `creators + matching content`
  - 再用 followers、engagement、location、gender、age、language 等筛选器细化
- 官方 FAQ 明确写到：
  - 它内建于 Modash 平台内
  - 当前工作在 Instagram
  - 用户先描述，再拿结果，再 refine with filters

## 关键摘录

- 官方说法强调“stop hunting by hashtags”与“type what you're looking for in your own words”。
- 结果层不是先让用户确认条件，而是先给“instant results”。
- AI Search 的落点仍是搜索工具，而不是独立 campaign copilot。

## 初步标注

- 🟢 亮点：最接近 `自然语言直搜 + 继续用筛选器收口`
- 🟢 亮点：`creators + matching content` 同屏，有助于从“抽象要求”过渡到“具体内容感知”
- 🟢 亮点：AI 感更多来自搜索体验升级，而不是强 chat UI
- 🔴 局限：当前官方明确工作在 Instagram，不是多平台统一搜索框
- ❓ 待确认：官方未强调多轮追问，更多是 `single prompt -> instant results -> manual refine`

## 对 Nox 的启发

- 适合作为 `搜索主入口本体升级` 的直接参考
- 说明“AI 在搜索里”不必等于“聊天流”
- 也说明自然语言搜索之后，显式筛选器仍然重要
