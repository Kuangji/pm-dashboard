## 核心事实

- HypeAuditor 官方首页已将 `AI search` 放在 `Discover influencers` 模块中。
- 官方 Discovery / Search 页面强调：
  - 支持 Instagram、TikTok、YouTube、X、Twitch 等
  - 220M+ profiles
  - 27+ filters
  - Audience Quality Score、location、audience demographics、influencer size 等筛选
- 官方博客同时公开了第二条 AI 路线：`AI Scout`
  - 通过 chat field/brief 找人
  - 根据 campaign brief 返回约 50 个 shortlist
  - 有 chat history
  - 可将结果加入 network / media plan / campaign
- 官方博客还明确说：
  - 除了独立 AI Scout，`AI Search inside Influencer Discovery` 也存在
  - AI Search 可以和 filters 叠加使用

## 关键摘录

- HypeAuditor 同时存在两条路线：
  - `Discovery 内 AI Search`
  - `独立 AI Scout`
- AI Scout 的语言很明确：不是搜索框升级，而是 `chatbot that helps you find influencers simply by chatting`
- Discovery 仍是典型的 filter-heavy 搜索工作台

## 初步标注

- 🟢 亮点：最完整地展示了“同一产品里并存两条 AI 路线”
- 🟢 亮点：AI Scout 对“brief -> shortlist -> 保存历史 -> 后续 campaign”非常完整
- 🟢 亮点：Discovery 内 AI Search + 27+ filters 的组合，适合专业用户
- 🔴 问题：如果直接照搬 AI Scout，会把 Nox 用户注意力从搜索框带走
- 🔴 问题：重筛选体系对新手仍然重
- ❓ 待确认：AI Search 在产品内的 UI 比例与默认入口权重

## 对 Nox 的启发

- HypeAuditor 说明“AI chat scout”并不等于最佳默认入口
- 对 Nox 来说，更合适的是借 Discovery 内 AI Search 的逻辑，避免走独立 scout
- 其 `AI Search + filters` 的双轨模式值得吸收，但要把追问压回搜索框内部
