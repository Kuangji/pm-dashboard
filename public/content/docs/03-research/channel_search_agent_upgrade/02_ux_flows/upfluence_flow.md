# Upfluence · UX Flow

用户目标：从 campaign idea 出发，借助 AI co-pilot 完成 brief、creator 选择与 campaign setup

入口触发：
- 入口位置：Jaice AI / Upfluence AI
- 前置条件：进入 Upfluence 平台

核心操作步骤：
1. 用户输入 campaign 目标
   - UI 反馈：Jaice 作为 co-pilot 接管 setup
   - 成功路径：系统帮助生成 brief、补偿建议与 creator search
2. 基于 marketplace data 搜 creator
   - UI 反馈：AI 用平台数据辅助找人
   - 成功路径：搜索不再是独立动作，而是 campaign setup 一部分
3. 进入后续 campaign 操作
   - UI 反馈：继续优化、管理、衡量

完成态：
- 成功：从 idea 到 campaign setup 的完整推进
- 取消：退出 co-pilot 流程

设计意图假设：
- 让 AI 不只“帮找人”，而是“帮把整个活动搭起来”

摩擦点：
- 搜索框本体不是主角
- 对单纯想快速搜频道的人来说范围过大

亮点：
- teammate 感强
- 适合作为语言风格与协同关系参考
