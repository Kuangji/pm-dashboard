# Klear · UX Flow

用户目标：通过 topic、受众画像与 performance metrics 找到适合品牌的 influencer

入口触发：
- 入口位置：Klear discovery/search
- 前置条件：进入 Klear 搜索工作台

核心操作步骤：
1. 用户使用 topic、hashtag、audience filters 发起搜索
   - UI 反馈：系统从 search index 中匹配 topic-aligned influencers
   - 成功路径：快速进入结构化筛选
2. 用户继续看 True Reach、demographics 等指标
   - UI 反馈：从 discovery 过渡到 vetting
3. 用户形成 shortlist

完成态：
- 成功：得到与品牌目标匹配的 influencer 集合
- 取消：继续改 topic 或 filters

设计意图假设：
- AI 应该主要体现在底层分类与排序，而不是前台话术
- 用户需要的是可靠的专业搜索感

摩擦点：
- 缺少更显性的自然语言入口

亮点：
- 工具感强
- 搜索与评估一体
