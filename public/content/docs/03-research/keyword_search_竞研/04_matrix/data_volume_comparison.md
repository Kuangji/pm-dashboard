## 三、表格版结论

### 结构性结论

| 观察点 | 结论 |
|---|---|
| 总量差距 | 表面上 Tarsight 约为 Nox 的 **39.17 倍**，但这个差距主要来自低粉长尾 |
| 低粉段（0-10K）| Nox 只达到 Tarsight 的 **2.08%**，说明长尾覆盖明显不足 |
| 有效段（10K+）| Nox 达到 Tarsight 的 **64.82%**，差距已显著收敛 |
| 最接近平台 | **YouTube**，10K+ 达到 **80.15%** |
| 主要短板 | **TikTok**，10K+ 仅达到 **58.51%** |
| Instagram 现象 | 总量差距极大，但 10K+ 已达到 **70.46%**，说明大头差距来自低粉尾部 |

### 业务解读

1. **如果比较“数据库体量”，Tarsight 明显更大。**
2. **如果比较“可用创作者库”，Nox 的真实差距远小于总量表现。**
3. **因此后续分析和汇报不能只用“总量”一个指标。**
4. **更合理的框架是：低粉长尾覆盖 vs 10K+ 有效创作者覆盖。**
5. **后续深挖优先级应是 TikTok 10K+ 召回逻辑，其次再评估 Instagram 低粉长尾是否值得补。**
