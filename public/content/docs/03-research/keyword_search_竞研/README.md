# 关键词搜索功能竞品调研

## 决策目标

对比 NoxInfluencer 与仟传 Tarsight 在「达人/内容关键词搜索 + 数据聚合分析」上的差距：

1. 同一关键词（Dreame）下，两边的**数据量差异**有多大？
2. 搜索结果中 **Badcase（不相关/错误匹配）** 的统计量差异如何？

**手动截图辅助模式** — 用户操作竞品 + 提供截图，AI 引导探索并结构化记录。
