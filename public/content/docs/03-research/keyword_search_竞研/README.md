# 关键词搜索功能竞品调研

## 决策目标

对比 NoxInfluencer 与仟传 Tarsight 在「达人/内容关键词搜索 + 数据聚合分析」上的差距：

1. 同一品牌关键词下，两边的**数据量与质量差异**有多大？
2. 搜索结果中 **Badcase（不相关/错误匹配）** 的统计量差异如何？

## 调研范围

| 项目 | 内容 |
|------|------|
| 竞品 | 仟传 Tarsight（强登录验证，手动截图模式）|
| 功能域 | 达人/内容关键词搜索 + 数据聚合分析 |
| 对比品牌 | Dreame（基准）+ Uwell（验证 A）+ Glad2Glow（验证 B）|
| 产出用途 | 独立调研报告（已完成，供分享与决策参考）|

---

## 研究方法

**决策背景**：NoxInfluencer 关键词搜索存在两个待评估问题——数据覆盖量是否达到竞品水平？搜索相关性是否有明显差距？本次调研通过与 Tarsight 横向对比量化差距，为功能优化提供数据支撑。

**采集方式**：手动截图辅助（用户在 Tarsight 内操作，AI 实时引导并结构化记录）。每次只给一个操作指令，等截图后再给下一步；无法截图的场景记录为 ❓ 未能观察。

**对比维度**：

| 维度 | 评估方式 |
|------|---------|
| 数据量 | 同词搜索结果总数（Tarsight vs Nox）|
| Badcase 率 | 抽样前 20 条，统计明显不相关条目数量 |
| 聚合视图 | 是否提供趋势图、品类分布、数据统计面板 |
| 交互体验 | 搜索入口、结果展示、筛选/排序能力 |
