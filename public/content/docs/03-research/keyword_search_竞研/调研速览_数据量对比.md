# 调研速览：Tarsight vs Nox 数据量对比

**主题**：Tarsight 与 Nox 在 YouTube / TikTok / Instagram 三平台的创作者规模对比  
**数据日期**：2026-03-25  
**原始数据**：`01_raw_data/tarsight数据量对比.md`

---

## 5 句结论版

1. **Tarsight 的总量优势主要来自 0-10K 的低粉长尾账号池，不代表在有效网红层面形成压倒性优势。**
2. 三平台总量对比中，Tarsight 约 **51.45 亿**，Nox 约 **1.31 亿**，Nox 仅为 Tarsight 的 **2.55%**；但这个差距几乎都由低粉段拉大。
3. 一旦只看 **10K+** 区间，差距显著收敛：Tarsight 约 **3911.6 万**，Nox 约 **2535.5 万**，Nox 已达到 Tarsight 的 **64.82%**。
4. 分平台看，**YouTube 最接近（80.15%）**，**Instagram 次之（70.46%）**，**TikTok 仍是主要短板（58.51%）**。
5. 因此，对外或对内判断时，不应直接拿“总库量”下结论，而应拆成 **低粉长尾覆盖** 与 **10K+ 有效创作者覆盖** 两个层面分别讨论。

---

## 一句话判断

**Tarsight 更像“超大盘全量账号池”，Nox 更接近“有效创作者库”；表面上的几十倍差距，核心来自低粉长尾，而不是高价值创作者层的全面落后。**
