# 头部平台定位图

## 定位图一：数据/分析深度 vs 执行工作流完整度

```text
高 数据/分析深度
↑
│          CreatorIQ
│      HypeAuditor        NoxInfluencer
│
│
│                    Upfluence
│               GRIN
│                    Modash
└────────────────────────────────→ 高 执行工作流完整度
```

### 解读

- **CreatorIQ**：更靠近企业级治理与分析套件
- **HypeAuditor**：更靠近真实性、分析与 vetting
- **NoxInfluencer**：更靠近数据厚度与平台能力广度
- **GRIN / Upfluence / Modash**：整体更靠近 workflow 侧
- **Modash**：在 workflow 区域里更轻、更偏中端市场

---

## 定位图二：企业重型程度 vs 中端 / DTC 友好度

```text
高 企业重型程度
↑
│      CreatorIQ
│               GRIN
│          Upfluence
│
│
│  HypeAuditor        NoxInfluencer
│                 Modash
└────────────────────────────────→ 高 中端 / DTC 友好度
```

### 解读

- **CreatorIQ** 最明显偏企业化
- **GRIN / Upfluence** 仍较重，但更直接服务品牌执行团队
- **Modash** 在 DTC 友好度和轻量化上最突出
- **Nox** 更像“能力很厚，但外部市场未必一眼看懂其 workflow 价值”的平台

---

## 最终定位语句

### Modash

**轻量一体化 creator workflow 平台，适合 DTC / 品牌团队把 creator marketing 当作持续运营渠道来跑。**

### NoxInfluencer

**能力更厚、平台更广、数据与监控更强的 influencer data platform，当前最需要的是把既有模块重新组织成更清晰的工作流价值。**

### 其他头部平台

- **CreatorIQ**：企业级总控
- **GRIN**：DTC creator relationship 与 ecommerce 深联动
- **Upfluence**：全栈 influencer + ecommerce + affiliate
- **HypeAuditor**：真实性、分析与 vetting 强项
