# 一页版速览

## 一句话定位

**Modash = 面向品牌 / 代理的 influencer discovery + workflow platform，强在“找到人之后还能顺手把合作跑完”。**

## 为什么它会被买单

- 搜索和筛选足够强
- 上手快，不像企业大平台那么重
- 把 Excel / Notion / 邮件 / tracking / 支付串成一个地方
- 支持响应快，用户感知明显

## 为什么它不是无敌的

- 数据库口径偏宽，仍需人工 vetting
- 对小团队来说并不便宜
- reporting、export、tracking、payments 的边角仍有抱怨
- 平台覆盖以 Instagram / TikTok / YouTube 为主，不是全平台型

## 它和 Nox 的真正差异

- **Modash 强**：workflow、DTC 场景、产品叙事、执行闭环
- **Nox 强**：数据与搜索底座、品牌监控与报告、平台广度、模块储备
- **关键不是“谁功能多”**，而是“谁把不同能力包装成更顺的购买理由”

## 头部格局里的位置

- **CreatorIQ**：重企业套件
- **GRIN**：重 DTC 关系管理
- **Upfluence**：重一体化电商与 affiliate
- **HypeAuditor**：重分析与真实性 vetting
- **Modash**：重轻量 workflow 与执行效率

## 最值得记住的判断

**Modash 的威胁不在于某个单功能点领先，而在于它把“发现 → 外联 → 追踪 → 支付”卖成了一条顺的工作流。**
